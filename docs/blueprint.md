# **App Name**: Kafas Build

## Core Features:

- Project Showcase: Display completed construction and furniture projects with images and descriptions.
- Service Catalog: List all available construction, building, furniture, and car services offered.
- Contact Form: Enable potential clients to submit inquiries and project requests.
- AI-Powered Design Assistant: The user uploads some information on what he wants. AI acts as tool for creating a preliminary furniture design concept based on client specifications. It can incorporate the existing portfolio, user preferences and information on the service catalog.

## Style Guidelines:

- Primary color: Deep slate blue (#2C3E50) for a professional and trustworthy feel.
- Background color: Light gray (#F4F4F8) to provide a clean and modern backdrop.
- Accent color: Burnt orange (#E34234) to highlight calls to action and important information.
- Body and headline font: 'Inter' for a clean, modern, and readable aesthetic.
- Use minimalist line icons to represent services and features.
- A grid-based layout will be implemented to ensure a structured, and responsive design, offering ease of navigation on all devices.
- Subtle fade-in animations for images and text to enhance user engagement.
